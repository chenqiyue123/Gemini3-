# 圣诞装饰互动体验

这是一个基于Three.js和MediaPipe的互动圣诞主题网页应用，通过手势控制可以展示不同的圣诞装饰效果，包括圣诞树、照片星系、照片特写和节日文字。

## 功能特点

- **手势控制**：通过手部动作与场景互动
- **动态效果**：包含雪花飘落、3D粒子动画和光影效果
- **照片互动**：支持上传个人照片，以多种形式展示
- **多模式切换**：四种不同的展示模式，各有特色

## 手势指令

| 手势 | 效果 |
|------|------|
| ✋ 张开手 | 照片星系爆发模式 - 照片将以星云状分布展示 |
| ✊ 握拳 | 修长悬浮树模式 - 粒子组成圣诞树形状 |
| ☝️ 单指 | 单张特写模式 - 随机展示一张上传的照片 |
| 👌 OK手势 | 巨型艺术字模式 - 显示"圣诞快乐"艺术字 |

## 使用方法

1. 打开网页，允许摄像头访问（用于手势识别）
2. 点击"📷 导入多张照片"按钮上传照片（建议10张以上效果更佳）
3. 使用不同手势控制场景切换

## 技术实现

- Three.js：用于创建3D场景和动画效果
- MediaPipe Hands：实现手势识别功能
- HTML5 Canvas：用于创建动态纹理和图形
- CSS3：实现UI样式和响应式设计

## 注意事项

- 确保设备有摄像头并授予访问权限
- 良好的光线条件有助于提高手势识别准确性
- 照片上传数量越多，星系模式效果越丰富

## 兼容性

支持现代浏览器（Chrome、Firefox、Safari等），建议使用最新版本以获得最佳体验。
